# PLAYCANVAS-3d

해당 저장소는 PlayCanvas를 사용한 웹 3D 개발에 대한 내용을 담고 있습니다.

[PLAYCANVAS](https://playcanvas.com) 는 WebGL기반의 [오픈 소스 게임 엔진](https://github.com/playcanvas/engine)으로, 웹 게임과 인터랙티브한 3D 콘텐츠를 웹 브라우저에서 제공합니다.
